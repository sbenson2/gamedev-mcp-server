MCP Tools
5 tools. Zero configuration.

Every tool is available the moment you connect. Search docs, start dev sessions, look up genre systems — all through natural language in your AI coding tool.

search_docs
TF-IDF full-text search across all game dev docs. Filter by category, module, or let it find the best match. Returns ranked results with relevance scores.
"search for camera smoothing techniques"
get_doc
Fetch a specific document by ID. Get the full content of any guide, reference, or concept doc instantly. Supports both numeric (G52) and slug (camera-theory) IDs.
"get doc G1" · "get camera-theory"
list_docs
Browse all available documentation organized by category and module. See what's available before diving in. Filter by core, engine-specific, or all.
"list all monogame guides"
session
Structured dev session co-pilot. Start a session, then use workflows: plan your architecture, make decisions, scope features, debug issues, or build new systems.
"start session" · "plan" · "debug"
genre_lookup
Instant genre-to-systems mapping. Input a genre (platformer, roguelike, metroidvania, etc.) and get back the required systems, recommended additions, and starter checklists.
"look up roguelike genre systems"
